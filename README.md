# 🎯 塔防游戏 (Tower Defense Game)

一个功能完整的 HTML5 2D 塔防游戏，使用纯 JavaScript 开发，无需任何依赖。

## ✨ 特性

- **4 种炮塔类型**，各有不同属性：
  - 🟢 **固定炮塔** - 向固定方向持续射击，简单可靠
  - 🟡 **旋转炮塔** - 自动瞄准最近敌人，360° 全方位
  - 🔴 **狙击炮塔** - 高伤害、射速慢、射程远
  - 🔵 **高速炮塔** - 射速极快，适合近距离防御

- **智能敌人系统**：
  - 轻甲、重甲、快速、Boss 4 种敌人类型
  - 难度随波次递增
  - 敌人血量、速度、奖励动态调整

- **完整游戏循环**：
  - 波次系统（最多 20 波）
  - 金钱和购买系统
  - 生命值系统
  - 得分统计

- **视觉效果**：
  - 粒子爆炸效果
  - 炮塔后坐力动画
  - 敌人血条
  - 拖拽高亮反馈
  - 瞄准指示器（旋转炮塔）

- **流畅体验**：
  - 60 FPS 游戏循环
  - 基于时间的更新
  - 响应式界面
  - 拖拽式炮塔部署
  - 炮塔位置交换

## 🕹️ 操作指南

| 操作 | 说明 |
|------|------|
| **拖拽炮塔** | 从左侧面板拖拽炮塔到战场 5×5 网格 |
| **移动炮塔** | 拖拽已部署的炮塔到其他格子交换位置 |
| **开始战斗** | 点击"开始战斗"按钮，敌人开始生成 |
| **停止战斗** | 再次点击按钮暂停游戏 |
| **购买炮塔** | 拖拽炮塔到空格子，自动扣除金钱 |

## 🎮 游戏机制

### 炮塔属性

| 炮塔 | 伤害 | 射速 | 射程 | 特点 |
|------|------|------|------|------|
| 固定炮塔 | 10 | 1.0s | 300 | 方向固定，简单耐用 |
| 旋转炮塔 | 8 | 0.8s | 250 | 自动瞄准，性价比高 |
| 狙击炮塔 | 50 | 2.0s | 400 | 高伤害，适合打 Boss |
| 高速炮塔 | 5 | 0.3s | 200 | 射速快，适合密集敌人 |

### 敌人波次

- **难度递增**：每波敌人数量和血量增加
- **敌人类型**：随波次解锁不同敌人类型
- **奖励系统**：击杀敌人获得金钱，完成波次有额外奖励

### 胜利条件

存活并击败所有 20 波敌人即可获胜！

## 🔧 代码亮点

### 1. 类型安全的配置对象
```javascript
const CONFIG = {
    GRID_SIZE: 5,
    TOWER_TYPES: { /* 详细配置 */ }
};
```

### 2. 模块化架构
- 独立类：Bullet, Tower, Enemy, Particle
- 清晰分离：更新循环与渲染循环

### 3. 性能优化
- 基于时间步长的更新（dt 参数）
- 对象池管理（bullet/enemy 数组）
- 最小化 DOM 操作
- 使用 requestAnimationFrame

### 4. 可扩展性
- 新增炮塔类型只需添加到 `CONFIG.TOWER_TYPES`
- 新增敌人类型只需添加到 `CONFIG.ENEMY_TYPES`
- 独立的碰撞检测系统

### 5. 用户体验
- 拖拽视觉反馈
- 炮塔后坐力动画
- 实时状态显示
- 智能路径待实现（可扩展）

## 🐛 已知问题 & 待优化

1. **路径寻找缺失**：敌人目前直线走向最近炮塔，未考虑障碍
2. **炮弹飞行碰撞**：使用简单距离检测，可优化为四叉树
3. **音效缺失**：可添加 Web Audio API 音效
4. **存储未实现**：本地保存最高分和波次记录
5. **移动端支持**：拖拽在触摸设备体验不佳，可改为点击选择+取消
6. **炮塔升级系统**：可添加升级、出售系统
