# Shooting Playground

一个基于 5x5 网格的肉鸽防御塔生存小游戏🎮

## 🎯 游戏核心玩法

### 🔄 回合制机制
1. **部署阶段**：在网格中放置防御塔，规划防御布局
2. **战斗阶段**：点击回合开始，怪物会从地图边缘出现并向核心移动
3. **结算阶段**：根据战斗结果获得金币和奖励，解锁新的炮塔类型

### 🛡️ 防御塔系统
- 多种炮塔类型：单发射击、AOE、减速、暴击等
- 炮塔可以升级，提升伤害、射程、攻速等属性
- 不同炮塔有不同的成本和适用场景

### 🧟 怪物系统
- 多波次怪物进攻，每波会更加强大
- 不同怪物类型：普通、快速、重型、飞行等
- 怪物会沿最优路径向核心移动

### 🎲 肉鸽元素
- 每次胜利后随机获得炮塔或升级选项
- 随机事件系统，带来增益或挑战
- 永久死亡机制，每一局都是全新体验

## 🗺️ 地图设计

- 初始地图：5x5 网格（可扩展）
- 核心区域：位于地图中心，需要保护
- 怪物出生点：地图四周边缘
- 路径系统：怪物会自动寻找最优路径

## 🎮 操作方式

- 鼠标点击：放置炮塔、升级炮塔、开始回合
- 键盘快捷键：快速选择炮塔类型、暂停游戏
